# AutoMSP Technology Stack

## Backend Architecture
- **Framework**: Node.js + NestJS
- **Database**: PostgreSQL + Redis
- **Message Queue**: RabbitMQ
- **API Gateway**: Kong

## AI & ML
- **LLMs**: Claude 3.5, GPT-4 Turbo
- **Vector DB**: Pinecone
- **Model Training**: PyTorch

## Infrastructure
- **Cloud**: AWS multi-region
- **Container**: Docker + Kubernetes
- **CI/CD**: GitHub Actions

## Monitoring
- **APM**: Datadog
- **Logging**: ELK Stack
- **Alerting**: PagerDuty

For detailed specifications, see Google Docs.
